# Navneet Raghav — Academic Homepage

This is a simple academic-style GitHub Pages homepage.

## How to publish

1. Create a new public GitHub repository named exactly:

   `navneet-raghav.github.io`

2. Upload these files to the repository root:

   - `index.html`
   - `style.css`

3. Go to repository **Settings → Pages**.

4. Under **Build and deployment**, choose:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Your site should appear at:

   `https://navneet-raghav.github.io`

If your GitHub username is different, rename the repository to:

`YOUR-GITHUB-USERNAME.github.io`
