## Important for your profile image
Your GitHub Pages website **cannot use a local Windows path** like:

`C:\Users\hp\Downloads\IMG_20260612_194726.jpg`

That path only works on your own laptop, not on the internet.

### What to do
1. Take your photo file `IMG_20260612_194726.jpg`
2. Rename it to **`profile.jpg`**
3. Upload it in the **same folder as `index.html`**
4. Push to GitHub

The page is already set to use `profile.jpg` automatically.

If `profile.jpg` is missing, it will show a placeholder image.
