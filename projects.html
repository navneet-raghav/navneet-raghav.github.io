Projects

TRACE-MultiStat: Internal Drift Audit for Small-Set Unlearning

Python, PyTorch, Hugging Face Transformers

Built a PyTorch based audit pipeline to study whether small forget sets still leave internal traces after approximate unlearning. I extract token-level hidden-state drift before and after unlearning, and use simple multi-statistic features for forget-vs-retain and forget-vs-unseen auditing.

I compared TRACE-MultiStat with output-only and compact TC-style baselines using DistilBERT and DistilRoBERTa on AG News, QNLI, and SST-2.

Deepfake Detection using CNN Ensemble and Grad-CAM

Python, PyTorch, OpenCV, Streamlit

Developed a deepfake detection system using XceptionNet and EfficientNet-B0 with ensemble learning. I also added Grad-CAM visualizations to understand which parts of the face the model is focusing on while predicting.

I handled class imbalance using weighted sampling and loss functions, and added augmentations like JPEG compression, noise, and blur to make the system more useful on real-world images.

Energy-Efficient Intrusion Detection

NIT Warangal | Smart Energy Systems | XGBoost, IoT Sensing

Working on an energy-efficient intrusion detection system for industrial use-cases. I developed an XGBoost based ML pipeline on a custom dataset to detect human heat signatures using a thermal array sensor.
